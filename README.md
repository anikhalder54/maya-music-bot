# Maya Music Bot

Maya is a simple and powerful Discord music bot designed to play songs, manage queues, and provide a smooth music experience inside your Discord server.

Features
- Play music from YouTube
-  Clear queue
-  Lightweight and fast performance
  
Setup Instructions
  1. Clone the repository
  2. git clone https://github.com/anikhalder54/maya-music-bot.git
  3. cd maya-music-bot
  4. Install dependencies
     npm install
  5. Create a .env file in the root directory and add
     DISCORD_TOKEN=your_bot_token_here
  6. Run bot
     node index.js
 
Bot Commands
 !play <song>  (Play a song from YouTube)
 !stop (Stop playback and clear queue)

Requirements
  Node.js v16+
  Discord Bot Token
  FFmpeg installed

Dependencies
  discord.js
  @discordjs/voice
  ytdl-core
  dotenv
